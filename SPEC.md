## Problem Statement

Pet owners often struggle to provide balanced, species-appropriate nutrition for their animals. Most people rely on guesswork or generic feeding guidelines printed on food packaging, which fail to account for individual factors like age, breed, weight, or allergies. Veterinary nutrition consultations are expensive and inaccessible for everyday meal planning needs.

---

## Desired Outcome

Users can input their pet's profile (species, age, weight, allergies, and health goals) and instantly receive a personalized 7-day meal plan with portion sizes and ingredient suggestions. Pet owners can make informed, confident feeding decisions without requiring a vet visit for routine nutrition questions.

---

## Must-Have Feature

> Given a pet's species, age, weight, and allergies, the app generates a safe and balanced 7-day meal plan using AI.

---

## Tech Stack

- **Frontend / UI:** Streamlit
- **Backend / Logic:** Python
- **AI Integration:** Claude or OpenAI API for meal plan generation
- **Data Storage:** CSV or session state (no database required)

## User Stories

### US-01 — Pet Profile Input
**As a** pet owner,  
**I want to** enter my pet's basic information (species, breed, age, weight, allergies, health goals),  
**so that** the app can generate a meal plan tailored to my pet's specific needs.

### US-02 — AI-Generated Meal Plan
**As a** pet owner,  
**I want to** receive a 7-day meal plan generated by AI based on my pet's profile,  
**so that** I have a structured, safe feeding schedule without consulting a vet.

### US-03 — Portion Size Guidance
**As a** pet owner,  
**I want to** see recommended portion sizes alongside each meal suggestion,  
**so that** I can avoid overfeeding or underfeeding my pet.

### US-04 — Allergy & Ingredient Safety
**As a** pet owner with an animal that has food sensitivities,  
**I want to** flag specific allergens or forbidden ingredients,  
**so that** the generated meal plan never includes unsafe foods for my pet.

### US-05 — Meal Plan Display & Export
**As a** pet owner,  
**I want to** view the weekly meal plan in a clean, readable format and optionally copy or download it,  
**so that** I can reference it throughout the week without reopening the app.

### US-06 — Multiple Pet Support (Nice-to-Have)
**As a** pet owner with more than one animal,  
**I want to** save and switch between multiple pet profiles,  
**so that** I can manage meal plans for all my pets in one place.

## Functional Specifications

### Page 1 — Pet Profile Form
- Input fields: species (dropdown), breed (text), age (number + unit), weight (number + unit), allergies/forbidden foods (multi-select or free text), health goals (e.g., weight loss, muscle gain, maintenance)
- Validation: required fields must be filled before submission
- "Generate Meal Plan" button triggers AI API call

### Page 2 — Meal Plan Results
- Display a 7-day table or card layout: Day → Meal → Ingredients → Portion Size
- Clearly highlight any substitutions made due to allergy constraints
- "Regenerate" button to request a new variation
- "Copy to Clipboard" or plain-text export option

### Page 3 — About / Tips (Optional)
- Brief guidance on pet nutrition basics
- Disclaimer: this app is not a substitute for professional veterinary advice

---

## Non-Functional Requirements

- App must load and display a generated meal plan within 15 seconds
- UI must be usable on both desktop and mobile browsers via Streamlit
- AI prompts must include safety guardrails (e.g., no toxic foods per species)
- Code must be hosted on GitHub with a public repository

---

## Out of Scope

- User authentication / login system
- Real-time database (no PostgreSQL, Firebase, etc.)
- Integration with pet food e-commerce or product recommendations
- Veterinary diagnosis or medical advice features
